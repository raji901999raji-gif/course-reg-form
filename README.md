# Volunteer Registration Web App

## Description
This is a simple web application adapted from a course registration form for NGO volunteer/training registration.

## Tech Stack
HTML, CSS, JavaScript 

## AI / Automation Idea
The app gives an automatic response after form submission.
Based on user input (area of interest), it suggests a suitable role like Healthcare or General Volunteer.

## Use Case
This can help NGOs like Jarurat Care to collect volunteer details and assign roles easily.
